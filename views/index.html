Introduction to API

This is a simple REST API with just 5 main endpoints. It lets you search branch and it's details of that branch which belongs to bank registered in India. We have accumulated records from 170 banks in total and about 10,000 branches in total. All this information is available with easy querying.


Usage

Here are the 5 endpoints of the API:

  • /branch/:$IFSC: replace the :$IFSC with your IFSC code and you will get address, city, district and other relevant details about the branch

  • /branches?bank=:$BANK_ID&city=:$CITY_NAME: replace the :$BANK_ID and :$CITY_NAME with Bank ID (make a request to /banks endpoint to know id) and City Name to get address, city, district and other relevant details about all the branches which belong to that bank in that city! You can use spaces in variables, test it on firefox!
